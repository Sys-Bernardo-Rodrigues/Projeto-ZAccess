Zaccess

Painel do dispositivo

Configuração

Conecte este dispositivo ao servidor Zaccess. As alterações exigem reinício do serviço.

Conexão com o servidor

Inclua a porta (ex.: 3001). Ex.: http://192.168.1.10:3001
Deve ser o mesmo cadastrado no painel do servidor.
Copie do painel web do servidor (Dispositivos → Editar → Copiar token).

Aplicar alterações

Após salvar a configuração, reinicie o serviço para aplicar.

sudo systemctl restart zaccess

O painel pode desconectar por alguns segundos após clicar. Se o botão não funcionar (sudo), execute o comando acima no terminal.
